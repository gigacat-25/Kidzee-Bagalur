# Kidzee Preschool Website Design Specification

## Design Philosophy

A playful, modern preschool website built around **bold curves,
oversized organic shapes, bright colors, and child-friendly
illustrations**. The design feels energetic while maintaining a premium
educational brand identity.

------------------------------------------------------------------------

# Brand Colors

  Color            Hex         Usage
  ---------------- ----------- ------------------------------
  Primary Purple   `#6F2C91`   Primary backgrounds, buttons
  Bright Yellow    `#FFD400`   Highlights, accents
  Light Lavender   `#ECE3F8`   Section backgrounds
  White            `#FFFFFF`   Cards, text contrast
  Dark Text        `#2C2C2C`   Body text

# Decorative Language

Use: - Organic blobs - Waves - Curves - Paper plane icons - Apple
outlines - Fish doodles - Dashed paths - Plus icons - Circular rings

Never use sharp edges.
